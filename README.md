# ParkVision 🚗

Веб-додаток для моніторингу вільних місць на парковці в реальному часі.

## Структура проєкту

```
parking_app/
├── app.py              # Flask бекенд
├── requirements.txt    # Залежності
├── CarParkPos          # Координати парко-місць (бінарний файл)
├── carPark.mp4         # Відео з камери
├── templates/
│   └── index.html      # Веб-інтерфейс
└── static/
    └── carParkImg.png  # Зображення парковки
```

## Запуск

1. Встановити залежності:
```bash
pip install -r requirements.txt
```

2. Запустити сервер:
```bash
python app.py
```

3. Відкрити браузер: **http://localhost:5000**

## Як це працює

- Flask стрімить оброблене відео у браузер (MJPEG)
- OpenCV аналізує кожен кадр: порогова фільтрація → підрахунок пікселів
- Якщо в зоні < 900 ненульових пікселів → місце вільне (зелена рамка)
- Статистика оновлюється кожну секунду через `/stats` API
